O questionário \emph{System Usability Scale} (SUS)~\cite{brooke1996} é composto por dez
afirmações, respondidas em uma escala Likert de cinco pontos (de ``discordo totalmente'' a
``concordo totalmente''). As dez afirmações originais, em tradução para o português, são:

\begin{enumerate}
    \item Eu acho que gostaria de usar este sistema com frequência.
    \item Considero o sistema desnecessariamente complexo.
    \item Achei o sistema fácil de usar.
    \item Acho que precisaria de ajuda de uma pessoa técnica para conseguir usar o sistema.
    \item Considero que as várias funções do sistema estão bem integradas.
    \item Acho que há muita inconsistência no sistema.
    \item Imagino que a maioria das pessoas aprenderia a usar o sistema rapidamente.
    \item Achei o sistema difícil de usar.
    \item Senti-me confiante ao usar o sistema.
    \item Precisei aprender muitas coisas antes de conseguir usar o sistema.
\end{enumerate}

Como descrito na Seção~\ref{sec:testes}, a aplicação do questionário SUS com usuários não foi
realizada nesta etapa do trabalho, ficando prevista como trabalho futuro
(Capítulo~\ref{cap:trabalhosfuturos}).
